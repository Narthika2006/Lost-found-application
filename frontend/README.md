# Frontend

## Development

- Install dependencies with `npm install`
- Start locally with `npm start`
- Build production assets with `npm run build`

## Environment

- Copy `frontend/.env.example` to `frontend/.env`
- Set `REACT_APP_API_URL` to your backend origin

## Notes

- The dashboard expects the backend `GET /api/items` endpoint to return paginated data
- The UI is optimized for a dark theme and production build output
